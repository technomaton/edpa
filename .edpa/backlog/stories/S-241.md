Closes the loop that caused D-17: commits reference tickets (CC scope) but nothing reconciles merge-evidence back to backlog status. New reconcile.py (script-first): walk merged commits on the main branch, extract item IDs from CC scopes/bodies, diff against backlog status, report items with delivery evidence still in pre-Done states (and Done items with zero evidence). Read-only suggestions by default; --apply flips status via the same transition rules (closed_at = last evidence commit date). Exposed as MCP read tool edpa_reconcile (#27) + /edpa:reconcile command + tests.
