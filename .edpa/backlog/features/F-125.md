Removed the `assignee` and `pr_author` evidence signal types end-to-end. They were sourced from GitHub issues/PRs and never fired in EDPA's local-first default flow (`run_gh` is a no-op without `EDPA_USE_GH=1`; the close-iteration path passes an empty issue_map). `commit_author` already credits the PR author's commits locally, making `pr_author` redundant.
