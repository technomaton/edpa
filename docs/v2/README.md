(the local-first re-architecture), shipped in 2.0.0 (2026-05) and refined
through 2.6.x. They are kept as the historical record of *why* V2 looks the
way it does; they are not maintained as current documentation and may
reference superseded details. For current docs see `docs/RUNBOOK.md`,
`docs/playbook.md`, `docs/methodology.md`, `docs/mcp.md`.
